# Website Delivery with Amazon CloudFront

## Project Overview
This project demonstrates how to host and deliver a static website using Amazon S3 and Amazon CloudFront. CloudFront improves website performance by caching content at edge locations around the world.

## AWS Services Used
- Amazon S3
- Amazon CloudFront

## Project Architecture
User → CloudFront → Amazon S3

## Steps Performed
1. Created an Amazon S3 bucket.
2. Uploaded the website files (HTML, CSS, and JavaScript).
3. Configured the S3 bucket.
4. Created a CloudFront distribution.
5. Selected the S3 bucket as the origin.
6. Waited for the distribution to deploy.
7. Accessed the website using the CloudFront domain.

## Features
- Static website hosting
- Fast content delivery using CloudFront
- Simple and scalable architecture
